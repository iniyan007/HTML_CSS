Welcome to My Website

This is a simple multi-page site built with only HTML and CSS.
No JavaScript — pages switch using the CSS :target trick!

See My Work →

How does this work?

Every "page" is just a <div> with an id. When you click a nav link like href="#work", the browser adds #work to the URL. CSS then uses the :target pseudo-class to show only that section.

Key CSS used here

.page — sets display: none on all pages.

.page:target — sets display: block on the active page.

@keyframes fadeIn — adds a gentle fade when switching pages.
