# Fresh Fruits Landing Page

## Overview

This project is a static landing page for a fictional brand called **Fresh Fruits**. The website showcases farm-fresh produce with a simple and responsive design using HTML and CSS.

## Features

* Semantic HTML structure using header, main, section, and footer
* Navigation bar with links to multiple pages
* Banner section with background image and overlay gradient
* Responsive design using CSS media queries
* Custom fonts integrated using Google Fonts

## Technologies Used

* HTML5
* CSS3
* Google Fonts

## Project Structure

```
project-folder/
│
├── index.html
├── style.css
├── background.jpg
```

## HTML Structure

### Header

The header contains the website title and a short description.

### Navigation Bar

The navigation bar includes links to:

* Home
* Shop
* About Us
* Contact

### Banner Section

* Displays a background image with a gradient overlay
* Includes main heading and tagline

### Footer

Contains copyright information.

## CSS Styling

### Global Styles

* Reset margin and padding
* Apply Poppins font to all elements

### Header

* Uses flexbox for alignment
* Space between title and description

### Banner

* Background image with linear gradient overlay
* Full viewport height

### Navigation Bar

* Center-aligned menu using flexbox
* Styled links as buttons with padding and border-radius

### Typography

* Custom fonts:

  * Poppins
  * Kaushan Script

### Footer

* Centered text with background color

## Responsive Design

The project uses media queries for different screen sizes:

### Max Width: 550px

* Reduced font sizes
* Navigation becomes vertical
* Smaller buttons

### Max Width: 350px

* Further reduced text sizes
* Compact layout adjustments

### Max Width: 250px

* Minimal font size for very small devices

## How to Run

1. Download or clone the project
2. Open `index.html` in any web browser

## Notes

* Ensure `background.jpg` is present in the same directory
* Navigation links currently point to static pages
